## 本地简单运行步骤
> pip install -r requirement.txt 
>>安装所有需要的配置文件

>运行redis服务
>>windows下直接下载redis，解压，点击redisserver.exe,运行起来即可

> python manage.py runserver 端口
>>将服务运行在本地进行简单的使用

> 在浏览器打开127.0.0.1：端口即可
>>服务跑起来后在浏览器看实现的效果
